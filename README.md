# 夕飯ルーレット

「今日の夕飯どうしよう」を10秒で終わらせる無料のWebアプリ(PWA)。

**アプリ本体: https://gryo1240.github.io/yuhan/**

- **🎲 ルーレット**: ボタンを押すと家庭料理36品からランダムに1品決定。名前がパラパラ切り替わる演出つき。「主役(お肉/お魚/麺・ごはん/卵・豆腐)」「調理時間(15/20/30分以内)」で候補を絞れる
- **🔮 気分診断**: 「こってり or さっぱり」「かける時間」「主役」の3問に答えると、おすすめベスト3を提案
- **決定したらそのまま作れる**: 材料(2人分)・調味料の分量・作り方つき。材料リストのコピーで買い物メモにも
- **決定履歴**: 直近3回に出た料理はルーレットに出にくくなるので、連日同じものになりにくい
- **結果をコピー**: 「今日の夕飯は『◯◯』に決定!🎲」をワンタップでコピーしてSNSや家族LINEへ
- データは端末内(localStorage)のみ。外部送信・登録不要

姉妹アプリ「週の献立プランナー」( https://github.com/gryo1240/kondate )・「のこりレシピ」( https://github.com/gryo1240/nokori )と同じ36品のレシピデータを使用。kondate=1週間まとめて計画 / nokori=ある食材から探す / **yuhan=今日の1食を即決**、という使い分けです。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JSすべて1ファイル。レシピデータ36品を内蔵) |
| `manifest.webmanifest` / `sw.js` | PWA化(インストール・オフライン対応) |
| `icon-*.png` / `apple-touch-icon.png` | アプリアイコン |
| `docs/screenshot-*.png` | 紹介用スクリーンショット |

`main` ブランチに push すると GitHub Pages に自動デプロイされます。

ブログ等での紹介記事を書く場合は [HANDOVER.md](HANDOVER.md) を参照してください。
